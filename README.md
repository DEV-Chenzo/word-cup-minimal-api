# word-cup-minimal-api

Esta é uma API mínima criada para estudo e prática com Node.js e TypeScript.

## Como funciona

A API expõe endpoints simples para consultar informações sobre seleções e campeões da Copa do Mundo. Ela lê dados armazenados localmente e retorna respostas em formato JSON.

## O que você encontra aqui

- Um servidor simples com rotas básicas
- Dados estáticos sobre times e vencedores
- Respostas em JSON para facilitar o aprendizado

## Como usar

1. Instale as dependências com `npm install`
2. Inicie o servidor com `npm run dev`
3. Acesse os endpoints disponíveis no navegador ou por ferramentas como Postman/Insomnia

## Objetivo do projeto

O projeto foi criado para ajudar no aprendizado de construção de APIs minimalistas, organização de código e uso de rotas em TypeScript.
